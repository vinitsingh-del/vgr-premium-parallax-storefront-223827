# VGR Premium Parallax Storefront

Static GitHub Pages-ready storefront for VGR-style grooming commerce.

## Pages

- `index.html` - conversion-focused home page
- `collection.html` - responsive product collection page
- `product.html` - VL-786 product detail page with gallery and sticky buy CTA

## CRO Elements

- Above-the-fold offer and product focus
- Sale anchoring with compare-at pricing
- Review, warranty, COD, replacement, and stock cues
- Sticky product CTA on the PDP
- Category rail and collection filters
- Desktop and mobile responsive product grids
- Motion-safe parallax layers with reduced-motion fallback

## Deploy On GitHub Pages

Use the repository root as the GitHub Pages source. The root `index.html` is the home page.
